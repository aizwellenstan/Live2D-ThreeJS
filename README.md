# Live2D_ThreeJS

Live2D WebGL SDKとThree.jsを連携するサンプルです。

[Live2D WebGL SDK](http://sites.cybernoids.jp/cubism-sdk2/webgl2-1)をDLして組み合わせれば使えます。
